# PortSwigger Web Security Academy - SQL Injection Writeups

## Overview

This repository documents my hands-on learning journey through the **SQL Injection** path on **PortSwigger Web Security Academy**. Each writeup covers the objective, methodology, payloads used, observations, impact, mitigation strategies, and key concepts learned while solving the labs.

The purpose of this repository is to strengthen my understanding of SQL Injection vulnerabilities, web application security, and secure coding practices through practical, lab-based exercises.
